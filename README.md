#  Sales Performance Analysis

This project provides a comprehensive end-to-end **Sales Performance Analysis** using raw sales data.  
It covers **data cleaning**,**Functions**, **exploratory data analysis**, **visualization**, and an interactive **Sales Performance Dashboard** built in Excel.

---

## Project Overview

The analysis was carried out to uncover performance trends, highlight key insights, and support data-driven decision-making.
It explores sales performance across regions, sales representatives, product categories, and seasonal patterns that influence revenue outcomes.

---

## Project Components

### **1. Data Cleaning**
- Standardized column formats  
- Removed duplicates and inconsistencies  
- Validated numeric fields and date formats  
- Handled missing values

### **2. Exploratory Data Analysis (EDA)**
Covers:
- Revenue distribution  
- Regional sales performance
- Product category performance  
- Sales representative contribution  
- Monthly and quarterly sales trends  

### **3. Sales Dashboard**
An Excel interactive dashboard that visualizes:
- Sales performace
- Sales representative analysis
- Regional insights
- Product and inventory metrics  
- Revenue by region
- Top-performing sales reps  
- Best and worst products  
- Seasonal trends  
- KPI summary and filters  

A screenshot is included in the `images/` folder.

---

##  Key Business Questions Answered

This project provides insights for the following strategic questions:

- Which regions generate the most revenue?  
- Who are the top-performing sales reps?  
- What are the best and worst performing products?  
- How do sales vary month-to-month and quarter-to-quarter?  
- What seasonal patterns impact sales performance?  

---

##  Strategic Recommendations

Based on the analysis and dashboard insights:

1. **Adjust pricing and discounts strategy** to reduce losses and improve margins. 
2. **Review and optimize cost structure** across all product categories.
3. **Improve inventory management** to reduce overstock and stockouts.  
4. **Improve product performance** by focusing on high-demand categories.
5. **Strenghten sales team performance** through better training and performance monitoring.
6. **Improve customer retention** by targeting returning customers and enhancing loyalty.
7. **Monitor profitability regularly** to track progress and support better decision-making.
